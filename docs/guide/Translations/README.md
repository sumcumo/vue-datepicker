# Translations

You can import only the translations that you need

```vue
<template>
  <DatePicker :language="en"></DatePicker>
</template>
<script>
import { en } from '@sum.cumo/vue-datepicker/dist/locale'

export default {
  data() {
    return {
      en: en,
    }
  },
}
</script>
```
